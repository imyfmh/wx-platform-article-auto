# Credentials File

Use YAML or JSON.

Required keys:

- `appId`
- `appSecret`

Optional keys:

- `forceRefresh`

YAML example:

```yaml
appId: wx1234567890abcdef
appSecret: 0123456789abcdef0123456789abcdef
forceRefresh: false
```

JSON example:

```json
{
  "appId": "wx1234567890abcdef",
  "appSecret": "0123456789abcdef0123456789abcdef",
  "forceRefresh": false
}
```
