# Topic Research

## Workflow

1. Clarify topic scope, region, and timeframe.
2. Run Brave Web Search first using the user topic/title as the primary query.
   - Default credentials source: `~/.pw`
   - Read Brave token from `[brave]` section, key `token` or `apiKey`
   - If token is missing, explicitly note that Brave search is unavailable and continue with best-effort source gathering
3. Collect 5–12 high-quality sources from search results (prefer primary sources).
4. Extract key points, numbers, dates, and opposing views.
5. Mark each point with confidence: high/medium/low.
6. Produce a research brief for drafting.
